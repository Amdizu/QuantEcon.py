QuantEcon

QuantEcon is a community based project that coordinates development of code for quantitative economic modeling. QuantEcon is sponsored and supported by the Alfred P. Sloan Foundation.

The quantecon Package

A stable release of the library is available through the Python package quantecon

Assuming you have pip running on your computer, you can install the latest stable release of quantecon by opening a terminal (or cmd on Windows) and typing

pip install quantecon

QuantEcon works well with the Anaconda Python distribution.

Documentation

Comprehensive documentation is currently being prepared.

At present, the majority of the code is described at quant-econ.net, in an on-line course authored by Thomas J. Sargent and John Stachurski

Developers

Development of QuantEcon is coordinated through our GitHub repository. If you'd like to get involved in development of the library please contact the project coordinators.
